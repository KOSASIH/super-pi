## 📋 Quick Start

```bash
git clone https://github.com/KOSASIH/super-pi.git
cd super-pi
npm install
npx hardhat compile

# Deploy PromptFactory v2.0
npx hardhat run scripts/deploy-prompt-factory-v2.js --network super-pi-l2

# Generate app batch (1,000 apps)
python3 scripts/app_genesis.py --batch 1000 --category "Digital Banking"

# Fiat rate sync
python3 scripts/fiat_rate_sync.py

# Run tests
npx hardhat test
```
